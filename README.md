# Text RPG

A browser-based text RPG game built with TypeScript and vanilla HTML/CSS.

## Gameplay
- Fight randomly spawned enemies (Goblin, Orc, Troll, Dragon)
- Each enemy has unique health and attack stats
- Use potions to heal during combat
- Survive as long as you can

## Features
- Random enemy spawning with spread operator copying to prevent state mutation
- Turn-based combat with randomized damage ranges
- Potion inventory system
- Live DOM updates on every action
- Full TypeScript with interfaces and type safety

## Tech Stack
- TypeScript
- HTML
- Vanilla JavaScript (compiled from TypeScript)

## What I Learned
I built this project to learn JavaScript and TypeScript fundamentals including DOM manipulation, interfaces, type assertions, array methods, and event listeners.
